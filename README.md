🔫 Russian Roulette – Симуляция русской рулетки на 7 языках
Адреналиновая игра-симуляция с реалистичной механикой револьвера.
Настройте количество игроков, патронов, режим прокрутки барабана и проверьте свою удачу.
Реализована на 7 языках программирования с единым интерфейсом, цветным выводом и расширенной статистикой.

🚀 Возможности
Настраиваемый револьвер – количество гнёзд (по умолчанию 6) и количество патронов (1–6).

Режимы игры:

classic – один патрон, барабан не крутится между выстрелами.

spin – барабан прокручивается перед каждым выстрелом (случайное положение).

duel – дуэль до первого выстрела (игроки ходят по очереди, пока кто-то не проиграет).

survival – выживание: один игрок против револьвера, каждый выстрел увеличивает шанс.

Многопользовательский режим – от 2 до 6 игроков (имена задаются).

Интерактивность – каждый ход игрок решает: выстрелить или передать ход (в некоторых режимах).

Статистика – количество сыгранных раундов, побед/поражений, процент выживаемости.

Сохранение рекордов – лучшие результаты сохраняются в файл.

Цветной вывод – выделение игроков, сообщений о выстреле, результатах.

Анимация – задержка для нагнетания напряжения (опционально).

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [режим] [опции]
Режимы
classic – классическая игра (один патрон, барабан не крутится между ходами).

spin – с прокруткой барабана перед каждым выстрелом.

duel – дуэль до первого выстрела.

survival – режим выживания (один игрок против револьвера).

Опции
Опция	Описание
-p, --players <N>	Количество игроков (2–6, по умолчанию 2)
-b, --bullets <N>	Количество патронов (1–6, по умолчанию 1)
-c, --chambers <N>	Количество гнёзд в барабане (по умолчанию 6)
-n, --names <name1,name2,...>	Имена игроков через запятую
-d, --delay <ms>	Задержка между действиями (для анимации)
-s, --stats	Показать статистику и выйти
-h, --help	Справка
Примеры
bash
# Классическая игра с 2 игроками (по умолчанию)
python roulette.py classic

# С прокруткой, 3 игрока, 2 патрона
python roulette.py spin -p 3 -b 2

# Дуэль с именами
python roulette.py duel -n "Алексей,Мария"

# Режим выживания с задержкой 500 мс
python roulette.py survival -d 500
Управление
Во время игры вводите y (да) или n (нет) для подтверждения выстрела (в некоторых режимах).

q – выход.

🧠 Логика игры
Барабан – массив из chambers элементов, часть из которых содержит патроны (bullets).

Положение – текущий индекс барабана, куда будет произведён выстрел.

Выстрел – если в текущей позиции есть патрон, игрок проигрывает; иначе патрон не срабатывает, и позиция сдвигается.

Прокрутка – в режиме spin положение барабана каждый раз выбирается случайно.

Ходы – игроки ходят по очереди; в режиме duel игра заканчивается после первого выстрела.

Выживание – один игрок делает выстрелы, пока не проиграет или не решит остановиться (с накоплением очков).

✨ Дополнительные фичи
Имена игроков – можно задать свои имена (по умолчанию Игрок 1, Игрок 2, …).

Статистика – сохраняется в ~/.roulette_stats.json и включает количество игр, побед, поражений.

Анимация – задержка между действиями для создания атмосферы.

Сценарий – игра предлагает игроку принять решение (выстрелить или передать ход) в некоторых режимах.
